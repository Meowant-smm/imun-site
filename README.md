# IMUN — Imaginary Model United Nations

A dark, diplomatic landing site for an imaginary Model United Nations conference built with HTML, CSS & JS.

## Features
- Live countdown timer to conference date
- Committee cards (IP, UNSC, UNGA, UNHRC)
- Gated registration + payment modal flow with QR code
- Dedicated Delegate Hub with MUN guides (SoP, GSL, Mod CoC, Unmod CoC, ROI)
- Modal popups with full examples for each MUN concept

## Built With
- HTML, CSS, JavaScript
- Designed via Stitch MCP
- Hosted on Netlify
